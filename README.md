# OA 進銷存系統

這是一個可直接開啟使用的靜態版 OA 進銷存系統，資料會儲存在瀏覽器 `localStorage`。

## 使用方式

1. 開啟 `index.html`
2. 點左側「範例資料」可快速載入示範商品、供應商、客戶與單據
3. 依序維護商品、往來單位、進貨單與銷貨單
4. 庫存會依進貨增加、銷貨扣減自動計算

## 手機 App 使用方式

此系統已支援 PWA，可用手機瀏覽器開啟線上網址後加入主畫面使用。

- iPhone / iPad：Safari 開啟網站，點分享，選擇「加入主畫面」
- Android：Chrome 開啟網站，點選選單，選擇「安裝應用程式」或「加入主畫面」

## 功能

- 總覽儀表板
- 商品資料管理
  - 銷售產品：紗號、支數、成分、色號、缸號、數量、單位與售價
  - 一般用品：品號、分類、進價、售價與安全庫存
- 供應商與客戶管理
- 往來單位管理：工廠類別、國內合作商類別、客戶名、E-MAIL、會計師與老闆欄位
- 進貨單新增與刪除
- 銷貨單新增與刪除
- 庫存明細與低庫存提醒
- 全域搜尋
- 貨品追蹤
- 生產流程看板
- 直播排程對應
- 出貨單管理
- 人員簿
- 請假申請
- 各分頁獨立報表與列印
